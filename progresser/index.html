Progresser

Weekend d'intégration : pour ceux qui voudraient chanter en choeur, mais ne se sentent pas en confiance pour le faire.

Atelier de technique vocale : ateliers ponctuels pour ceux qui voudraient s'améliorer encore

Atelier de rythmique : apprendre à vivre le rythme dans son corps

Cours individuels : pour tous ceux qui souhaitent développer leur technique vocale.

Les petits ensembles : vous voulez gagner en autonomie dans le chant chorale ? Nous vous proposons un travail à petits effectifs afin de progresser plus vite.
